プロジェクト課題で作成したページ: project.html

数当てゲーム

今回作成した数あてゲームでは、javascriptを用いてユーザーの入力情報から処理を行っている。 1~10の乱数を正解の値に設定する。ユーザーがテキストボックスに数字を入力し、ボタンを押すと予想が正しいか、予想より高い、低いと判定するようにした。 また、入力回数を変数で管理し、画面に予想回数の入力とその回数に応じて異なった処理を行っている。 表示される内容は、予想が正しいか、間違っていれば答えが大きいか小さいか、ゲーム終了後のメッセージなどを条件分岐で表示している。

プロジェクト課題で作成したページ: kazuate.html

各授業回の学習事項

第1回 WWW と HTML

第1回講義では、HTMLの文書の作成方法について学びました。 ただ文章を表示させるだけでなく、文字の大きさ、リンクの設置、特殊文字の表示、リストを作成など、Webページを作るのに必須な工程を身に着けました。

演習の成果: 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

第2回ではCSSを用いてHTMLで制作したページに色や装飾を加え、Webページの見た目を変更する方法を学びました。 CSSでは装飾だけでなく、webページを見やすくする変更ができるため、webページ作成には必要だと感じました。

演習の成果: 演習2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

第3回では、JavaScriptの構文やコンソールに文字を表示させる方法を学びました。for-of構文を使うことでデータの量が多くても簡単に管理することができます。 これを使い、Webページに便利な機能を搭載することができると考えました。

演習の成果: 演習3-6: オブジェクトの参照

第4回 DOM と JavaScript

第4回講義では、Javascriptを用いてボタンや入力フォームなどのイベント処理をし、入力値の取得や画面表示などの方法について学びました。 また、数当てゲームを通して全体のイベント処理の流れやプログラムの構成を理解しました。

演習の成果: 演習4-10: 足し算ページ
